## Descripción del Proyecto
Desarrollo de un sistema en modo consola utilizando **JavaScript** y **Node.js**. El objetivo es gestionar el ciclo de vida de los préstamos de libros, siguiendo una estructura similar a la de un sistema de ventas (Facturación):

* **Cabecera:** Registro de socio y datos del préstamo.
* **Detalle:** Carga de libros y cantidades.
* **Lógica/Pie:** Cálculo de subtotales de ejemplares y gestión de multas por mora.

## Tecnologías Utilizadas
* JavaScript (Runtime: Node.js)
* Git / GitHub (Control de versiones)
* SQLite (Planificado para etapas posteriores)
